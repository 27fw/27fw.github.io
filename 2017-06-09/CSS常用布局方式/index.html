解决浮动元素溢出问题

  • 在外围div底部添加一个<br />然后设定br {clear: both};（需要添加额外的html）。

  • 将外围div设定为float（但是容器后面的的元素要设定clear）

  • 将外围块设定为overflow: hidden;， overhidden会强制外围块放大到包含浮动元素，如果是ie6及之前的浏览器，再添加zoom: 1;不过如果div中有绝对定位的元素时，可能显现不出来。
